# 5E Lesson Plan Template[^1]

Use this template as the front-facing teacher document.

Keep this file concise. Put long, step-by-step teaching explanations in the companion tutorial.

## Course

| Course code and course name (for this repo: ICD2O Digital Technology and Innovations in the Changing World). |
| :---- |

## Teacher

| Teacher name(s). If co-teaching, list all teachers. |
| :---- |

## Lesson Number

| Lesson sequence number in `NN` format (for example `00`, `01`, `02`). |
| :---- |

## Title

| Clear, student-friendly lesson title aligned with the main task or concept. |
| :---- |

## Curriculum Expectations

| Use a bulleted list with one expectation per bullet. Use exact codes/wording from `resources/ICD2O_2023.md`. |
| :---- |

## Learning Goals

Students will be able to:

| Use a bulleted list with one measurable goal per bullet. Keep goals observable and lesson-sized. |
| :---- |

## Materials/Resources

| Use a bulleted list with one item per bullet. Include links/paths for the lesson plan, tutorial, notebook, assignment, datasets, and any slides/handouts. |
| :---- |

## Lesson Body

| 3-6 sentence overview of the lesson arc and timing. Explain how the class moves through `Engage -> Explore -> Explain -> Elaborate -> Evaluate`. |
| :---- |

### Engage

| Hook and activation of prior knowledge. Include time estimate, teacher moves, student actions, and success signal. |
| :---- |

### Explore

| Hands-on investigation or guided activity. Include tools/data used, student workflow, and checkpoints. |
| :---- |

### Explain

| Direct teaching and meaning-making. Define key vocabulary, connect to student observations, and clarify misconceptions. |
| :---- |

### Elaborate

| Transfer/extension task where students apply learning in a new context. Include differentiation options if needed. |
| :---- |

### Evaluate

| Evidence of learning gathered during and/or after class. Include what is assessed, how, and criteria for success. |
| :---- |

## Homework

| Optional follow-up task. State exactly what students do, what they submit (if anything), and due timing. |
| :---- |

## Comments

| Teacher-facing implementation notes: pacing adjustments, accommodations, grouping, tech setup, safety/logistics, and what to revise after teaching. |
| :---- |

[^1]:  [Why We Chose the 5E Lesson Plan Format](https://docs.google.com/document/d/1oTrJQprVIA7ggm6wgOXY9Z9adnE6T-9atT3vzn68zT0/edit?tab=t.0#heading=h.eh15dw7kt3ky)
