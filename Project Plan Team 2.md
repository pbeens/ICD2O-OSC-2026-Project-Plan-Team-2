## 2. Project Overview

**Project Title:**
ICD2O Beginner Data Analysis Pathway (Lessons 00-02)

**Purpose/Rationale:**
This project builds a beginner-friendly path into data analysis using authentic Canadian open data. It addresses the need for students to understand not only how to run code, but why analysis matters for evidence-based decisions. The pathway starts with conceptual foundations (data analysis and IPO model), then tool fluency (Jupyter Notebook workflow), and then applies those skills in a full real dataset lesson (extreme heat events).

**Student-Friendly Learning Goals:**
- I will learn to explain what data analysis is and why it matters.
- I will be able to use the IPO model (Input, Process, Output) to describe a data workflow.
- I will be able to use Jupyter Notebook cells (markdown + code) and run them in order.
- I will be able to clean and analyze a beginner-level dataset using Python.
- I will demonstrate evidence-based conclusions from a chart.

## 4. Defined Scope

### Students Will:
- Build conceptual understanding of data analysis using IPO.
- Develop procedural fluency with Jupyter Notebooks.
- Perform beginner-level data loading, cleaning, and visualization.
- Interpret chart output to support simple decisions and claims.

### Students Will Not:
- Use advanced statistics or machine learning in this sequence.
- Use ZIP-based ingestion in beginner notebook workflow.
- Be graded on highly complex multi-file programming structures.
